# QA Engineer — Портфолио проектов
 
Инженер по тестированию. Ручное тестирование веб- и мобильных приложений, тестирование API, работа с базами данных.
 
В репозитории собраны учебные и pet-проекты: тестовая документация (чек-листы, тест-кейсы), баг-репорты и отчёты о тестировании.
 
## 🧰 Стек и навыки
 
**Тестирование:** функциональное, регрессионное, кроссбраузерное, тестирование вёрстки, ретест дефектов, тестирование мобильных приложений (Android), тестирование REST API
 
**Тест-дизайн:** классы эквивалентности, граничные значения, анализ требований, чек-листы, тест-кейсы, баг-репорты
 
**Инструменты:** Postman, SQL, DevTools, Android Studio, Apidoc, Figma, Google Таблицы
 
**Технологии:** Python, REST API, HTTP, клиент-серверная архитектура
